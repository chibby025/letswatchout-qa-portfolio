# BUG-017: WASD Camera Controls Active During Chat Input

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-017 |
| **Severity** | Medium |
| **Priority** | P2 |
| **Feature Area** | 3D Cinema / Input Handling |
| **Status** | ✅ Fixed & Verified |

---

## Description

When typing in chat, WASD keys move camera instead of typing letters, making chat unusable in 3D cinema.

---

## Root Cause

Keyboard event listeners not checking input focus state.

---

## Fix Applied

```jsx
const handleKeyDown = (e) => {
  const isTyping = ['INPUT', 'TEXTAREA'].includes(
    document.activeElement.tagName
  );
  if (isTyping) return;  // Ignore WASD when typing
  
  if (e.key === 'w') moveForward();
  if (e.key === 'a') moveLeft();
  if (e.key === 's') moveBackward();
  if (e.key === 'd') moveRight();
};
```

---

## Verification

✅ Typed "wasd" in chat - no camera movement  
✅ Closed chat, pressed WASD - camera moved

---

**Status:** ✅ Fixed
