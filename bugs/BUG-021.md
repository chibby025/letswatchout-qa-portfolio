# BUG-021: Fullscreen Performance Degradation

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-021 |
| **Severity** | Medium |
| **Priority** | P2 |
| **Feature Area** | 3D Cinema / Performance |
| **Status** | ✅ Fixed & Verified |

---

## Description

Entering fullscreen mode causes frame rate to drop from 60fps to 15fps.

---

## Root Cause

Renderer resolution not optimized - rendering at 4K on 1080p displays.

---

## Fix Applied

```jsx
const handleFullscreen = () => {
  const width = window.screen.width;
  const height = window.screen.height;
  
  const maxWidth = Math.min(width, 1920);
  const maxHeight = Math.min(height, 1080);
  
  renderer.setSize(maxWidth, maxHeight);
  renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2));
};
```

---

## Verification

✅ Entered fullscreen on 4K display  
✅ Frame rate: 60fps (consistent)

---

**Status:** ✅ Fixed
