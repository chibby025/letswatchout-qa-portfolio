# BUG-019: Ending LiveShare Changes User's Seat

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-019 |
| **Severity** | Medium |
| **Priority** | P2 |
| **Feature Area** | LiveShare / 3D Cinema |
| **Status** | ✅ Fixed & Verified |

---

## Description

When user stops screen sharing, their 3D cinema seat position randomly changes.

---

## Root Cause

LiveShare end handler incorrectly resets seat state, triggering random seat assignment.

---

## Fix Applied

```jsx
const endLiveShare = async () => {
  await room.localParticipant.unpublishTrack(screenTrack);
  // Removed incorrect line: setSeat(null);
  setIsSharing(false);
};
```

---

## Verification

✅ Selected seat B3  
✅ Started screen share  
✅ Ended screen share  
✅ Still in seat B3

---

**Status:** ✅ Fixed
