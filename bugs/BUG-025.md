# BUG-025: DOB Requirement Not Enforced

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-025 |
| **Severity** | Medium |
| **Priority** | P2 |
| **Feature Area** | Authentication / Validation |
| **Status** | ✅ Fixed & Verified |

---

## Description

Users can skip date of birth during registration, bypassing age verification.

---

## Root Cause

Registration form validation allows null DOB values.

---

## Fix Applied

```jsx
const validateForm = () => {
  if (!formData.dateOfBirth) {
    setError("Date of birth is required");
    return false;
  }
  
  const age = calculateAge(formData.dateOfBirth);
  if (age < 13) {
    setError("You must be at least 13 years old");
    return false;
  }
  
  return true;
};
```

**Backend validation:**
```go
if user.DateOfBirth.IsZero() {
    return errors.New("date of birth is required")
}
```

---

## Verification

✅ Attempted registration without DOB  
✅ Error: "Date of birth is required"  
✅ All new users have valid DOB

**Status:** ✅ Fixed
