# BUG-014: Screen Share Audio Not Working

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-014 |
| **Severity** | High |
| **Priority** | P1 |
| **Feature Area** | Screen Share / LiveKit |
| **Status** | ✅ Fixed & Verified |

---

## Description

When users share screen with system audio, participants hear no audio - only see video feed.

---

## Root Cause

LiveKit track subscription not including audio track for screen share.

---

## Fix Applied

```jsx
room.on(RoomEvent.TrackSubscribed, (track, publication, participant) => {
  if (publication.source === Track.Source.ScreenShare) {
    if (track.kind === Track.Kind.Video || track.kind === Track.Kind.Audio) {
      track.attach(elementRef.current);
    }
  }
});
```

---

## Verification

✅ Shared screen with YouTube video  
✅ Participants heard audio + saw video

---

**Status:** ✅ Fixed
