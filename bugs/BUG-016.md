# BUG-016: Join Now from "Watching Now" Not Working

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-016 |
| **Severity** | High |
| **Priority** | P1 |
| **Feature Area** | Navigation / Session Discovery |
| **Status** | ✅ Fixed & Verified |

---

## Description

"Join Now" button on "Watching Now" page returns 404 error instead of joining active session.

---

## Root Cause

Session ID not passed to join endpoint - undefined variable in onClick handler.

---

## Fix Applied

```jsx
// Before: sessionId undefined
onClick={() => joinSession(sessionId)}

// After: Use session.id from map
onClick={() => joinSession(session.id)}
```

---

## Verification

✅ Clicked "Join Now" on 5 active sessions  
✅ All joined successfully

---

**Status:** ✅ Fixed
