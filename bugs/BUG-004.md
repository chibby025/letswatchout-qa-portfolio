## Description

Users were automatically logged out every 5-30 seconds in an infinite loop, making the platform completely unusable.

**Business Impact:**
- Platform unusable (showstopper bug)
- All 8 beta users affected
- Emergency fix required within 24 hours

---

## Root Cause

**Location:** `backend/middleware/auth_middleware.go` - Line 62  
**Issue:** JWT validation compared `IssuedAt` instead of `ExpiresAt`

```go
// BUG: IssuedAt is ALWAYS in the past!
if claims.IssuedAt < time.Now().Unix() {
    return nil, errors.New("token expired")
}
```

---

## Fix Applied

```go
// FIXED: Compare ExpiresAt (future timestamp)
if claims.ExpiresAt < time.Now().Unix() {
    return nil, errors.New("token expired")
}
```

---

## Verification Steps

✅ Logged in at 10:00 AM  
✅ Used platform for 2 hours: No logouts  
✅ Waited 24 hours: Token correctly expired  
✅ All 8 beta users tested: Zero logout loops
