# BUG-012: Instant Watch Redirecting to Wrong Watch Type

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-012 |
| **Severity** | High |
| **Priority** | P1 |
| **Feature Area** | Navigation / Session Types |
| **Status** | ✅ Fixed & Verified |

---

## Description

"Instant Watch" button redirects users to lecture hall instead of video watch room.

---

## Root Cause

Routing logic uses incorrect room type parameter.

---

## Fix Applied

```jsx
// Before: Wrong room type
navigate(`/session/${sessionId}?room=lecture-hall`);

// After: Correct room type
navigate(`/session/${sessionId}?room=videowatch`);
```

---

## Verification

✅ Clicked "Instant Watch" on 10 content items  
✅ All redirected to videowatch correctly

---

**Status:** ✅ Fixed
