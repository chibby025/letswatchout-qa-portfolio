# BUG-018: Seat Map Doesn't Persist on Refresh

| Field | Value |
|-------|-------|
| **Bug ID** | BUG-018 |
| **Severity** | Medium |
| **Priority** | P2 |
| **Feature Area** | 3D Cinema / State Management |
| **Status** | ✅ Fixed & Verified |

---

## Description

User's selected seat position in 3D cinema resets on page refresh, requiring re-selection.

---

## Root Cause

Seat selection stored only in React state, not in database.

---

## Fix Applied

```jsx
const selectSeat = async (seatId) => {
  await axios.post('/api/sessions/seat', {
    session_id: sessionId,
    user_id: userId,
    seat_id: seatId
  });
  setSelectedSeat(seatId);
};

useEffect(() => {
  const fetchSavedSeat = async () => {
    const response = await axios.get(`/api/sessions/${sessionId}/user-seat`);
    if (response.data.seat_id) {
      setSelectedSeat(response.data.seat_id);
    }
  };
  fetchSavedSeat();
}, [sessionId]);
```
