# Google Sheets Test Case Template

**Quick Start:** 5-minute setup to professional test management system

---

## 📥 Setup Instructions

1. **Download** `test_cases_template.csv` from this folder
2. **Go to** [Google Sheets](https://sheets.google.com)
3. **Click** File → Import → Upload → Select CSV
4. **Follow** detailed setup in `GOOGLE_SHEETS_SETUP_GUIDE.md`
